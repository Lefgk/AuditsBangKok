## Project Description

Lemonad is a comprehensive DeFi ecosystem on Monad featuring:

### 1. Decentralized Exchange (DEX)
- Uniswap V2 fork with optimized gas efficiency
- Factory: `0x0FEFaB571c8E69f465103BeC22DEA6cf46a30f12`
- Multiple trading pairs including LEMON/WMON, MON/USDC

### 2. Yield Boost Vault
- Single-sided LEMON staking
- Contract: `0x749F5fB1Ea41D53B82604975fd82A22538DaB65a`
- Currently: 3,182,156 LEMON staked

### 3. LemonChef Farming
- MasterChef-style yield farming
- Contract: `0x09C0B23c904ec03bFbf8B20686b4a42add71ad6a`
- Multiple LP farming pools

### 4. iLemonati NFT Collection
- Phased minting (OG, Whitelist, Public)
- Utility NFTs for ecosystem benefits

### 5. Gaming Suite
- On-chain dice games
- Lottery system
- PvP battles
- Racing games
